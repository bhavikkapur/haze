# HAZE

### Turning environmental data into accountability.

An open engineering initiative developing low-cost sensing systems, embedded electronics, and intelligent software to identify pollution sources and generate evidence for environmental accountability.

</div>

---

## Overview

Air quality monitors can tell us **how polluted the air is**.

Very few can tell us **where that pollution came from**.

HAZE is an ongoing engineering project exploring how embedded systems, environmental sensing, machine learning, and geospatial data can be combined to improve particulate matter source attribution and strengthen public accountability.

This repository documents every stage of that journey—from the first idea and early rejections to prototype development, field testing, and future deployments.

---

## Vision

Build an open platform capable of:

- Monitoring particulate matter in real time
- Improving source attribution using sensor fusion and machine learning
- Generating evidence-based environmental reports
- Making pollution data accessible and actionable
